# vscode-extension-shell-shortcut

## development

```sh
pnpm i
npm run dev
npm run prettier
npm run test
cp -r .vscode-example/. .vscode  # then hit F5 to open src/test/demo-workspace

# publish to marketplace
volta install vsce  # install globally
npm run package
```
